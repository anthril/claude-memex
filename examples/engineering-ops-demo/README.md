# engineering-ops demo

The `engineering-ops` profile's template tree is the reference scaffolding — there is no separate demo here, because `/memex:init engineering-ops` produces exactly what you see under [`../../templates/profiles/engineering-ops/`](../../templates/profiles/engineering-ops/).
